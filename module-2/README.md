# Module 2
A simple WebGL program that demonstrates a 2D Sierpinski Gasket.

## Usage
1. Ensure you are in the correct directory of the sub-project.
2. Run ```src/gasket.html``` file in a browser of your choice.
3. Enjoy the program(s).
