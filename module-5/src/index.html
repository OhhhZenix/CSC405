Tips: Use 'x' to interact with left and right values. Use 'y' to interact with bottom and top values. Use 'z' to interact with near and far values. Use 'r' to interact with radius value. Use 'o' to interact with theta value. Use 'p' to interact with phi value. Use 'shift' plus the other keys to go in the opposite direction of the values.
